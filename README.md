# AI Outfit Recommendation System (OOP Based)

An intelligent, Object-Oriented Programming (OOP) based recommendation engine that suggests clothing outfits and accessories based on user preferences, occasions, and color harmony principles.

## 🚀 Key Features
* **OOP Architecture:** Fully built using clean Object-Oriented principles (Inheritance, Encapsulation, and Polymorphism).
* **Color Harmony Logic:** Suggests matching clothing combinations based on complementary and analogous color theory.
* **Occasion-Based Filtering:** Tailors outfits dynamically for Formal, Casual, Party, or Sports wear.
* **Interactive Scoring:** Features an internal rating system to learn and adapt to user style preferences over time.

## 🛠️ Technologies Used
* **Language:** Python 3.x
* **Concepts:** Object-Oriented Programming (OOP), Functional Programming, Decorators
